# Ağ Temelleri

## IP Adresi

IP adresi cihazların internetteki adresidir.
Bilgisayarlar birbirini IP adresi ile bulur.
Bir siteye girdiğimiz zaman bilgisayar o sitenin IP adresine bağlanır.

## Port

Port bilgisayardaki uygulamaların bağlantı noktasıdır.
Aynı IP adresinde farklı programlar farklı portları kullanabilir.
Örneğin web siteleri genelde 80 ve 443 portlarını kullanır.

## DNS

DNS site isimlerini IP adresine çevirir.
Biz google.com yazarız ama bilgisayar bunu IP adresi olarak kullanır.
DNS sayesinde IP adreslerini ezberlememize gerek kalmaz.

## TCP

TCP verilerin güvenli şekilde gönderilmesini sağlar.
Veri eksik giderse kontrol eder ve tekrar gönderilmesini sağlar.
Web siteleri ve dosya indirme işlemlerinde kullanılır.

## UDP

UDP daha hızlıdır ama TCP kadar güvenli değildir.
Verinin eksik gidip gitmediğini çok fazla kontrol etmez.
Oyunlarda, canlı yayınlarda ve sesli görüşmelerde kullanılır.

## Paket

İnternette veriler küçük parçalara bölünerek gönderilir.
Bu parçalara paket denir.
Mesela bir dosya tek parça gitmez, paketler halinde karşı tarafa gider.

## Ping

Ping bağlantıyı kontrol etmek için kullanılır.
Bir cihaza ulaşılıp ulaşılamadığını gösterir.
Ayrıca bağlantının kaç milisaniyede cevap verdiğini gösterir.

## Traceroute

Traceroute verinin hedefe giderken hangi yollardan geçtiğini gösterir.
Bağlantının nerede yavaşladığını anlamaya yardımcı olur.

## Nslookup

Nslookup DNS kontrolü yapmak için kullanılır.
Bir site adının hangi IP adresine gittiğini gösterir.

## Sonuç

Ağ temelleri bilgisayarların internette nasıl haberleştiğini anlamamızı sağlar.
IP, port, DNS, TCP ve UDP yazılımın internet üzerinde nasıl çalıştığını öğrenmek için önemlidir.
