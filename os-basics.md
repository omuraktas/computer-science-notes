# İşletim Sistemi Temelleri

## Kernel (Çekirdek)

Kernel işletim sisteminin en önemli kısmıdır.
Donanım ile yazılım arasında bağlantı kurar.
Programların CPU, RAM, disk gibi parçaları kullanmasını sağlar.

## Process (Süreç)

Process çalışan program demektir.
Bir program açıldığında işletim sistemi onu process olarak çalıştırır.
Her process bellekte yer kaplar.

## Thread (İş Parçacığı)

Thread process içinde çalışan küçük görevdir.
Bir process içinde birden fazla thread olabilir.
Thread’ler aynı program içinde farklı işleri yapmaya yarar.

## Bellek Yönetimi

Bellek yönetimi RAM’in düzenli kullanılmasını sağlar.
İşletim sistemi programlara bellekte yer ayırır.
Program kapanınca kullanılan bellek geri alınır.

## CPU Zamanlayıcıları

CPU zamanlayıcıları işlemcinin hangi programı çalıştıracağını belirler.
İşletim sistemi CPU zamanını programlar arasında paylaştırır.
Bu sayede bilgisayar aynı anda birden fazla işlem yapıyormuş gibi çalışır.
